# 🦅 Quiz Sejarah Indonesia

Aplikasi quiz interaktif berbasis web untuk menguji pengetahuan Sejarah Indonesia.
Dibuat dengan HTML, CSS, dan JavaScript murni — tanpa framework, tanpa install!

## ✨ Fitur
- 📜 15 soal tersedia, dipilih 10 secara acak setiap sesi
- ⏱ Timer 20 detik per soal
- ✅ Feedback langsung + penjelasan setiap jawaban
- 🏆 Rekap skor dan akurasi di akhir quiz
- 📱 Responsive — bisa dimainkan di HP maupun PC

## 🚀 Cara Menjalankan
Buka langsung di browser:
👉 https://username-kamu.github.io/quiz-sejarah-indonesia

Atau download file index.html lalu buka di browser lokal.

## 🛠 Teknologi
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📚 Topik Soal
Proklamasi, Sumpah Pemuda, tokoh pahlawan, perjanjian kemerdekaan,
peristiwa bersejarah, dan era reformasi Indonesia.
